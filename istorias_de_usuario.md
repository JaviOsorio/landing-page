### 1. Historia de Usuario – Sección Inicio (Hero)
Como visitante nuevo,
quiero entender de inmediato qué hace el sistema de pesaje,
para decidir si vale la pena seguir leyendo o pedir información.

**Criterios de aceptación:**
- Debo ver un título claro y atractivo.
- Un texto rápido que explique el sistema.
- Un botón para pedir información.


### 2. Historia de Usuario – Sección Características
Como persona interesada en soluciones de pesaje,
quiero ver rápidamente qué ofrece el sistema,
para saber si tiene lo que necesito.

**Criterios de aceptación:**
- Debo ver al menos tres características explicadas.
- Cada característica debe estar en una tarjeta visual.
- Todo debe ser fácil de leer.


### 3. Historia de Usuario – Sección Beneficios
Como usuario que busca mejorar procesos,
quiero conocer los beneficios reales del sistema,
para evaluar si vale la pena implementarlo.

**Criterios de aceptación:**
- Lista clara de beneficios.
- Cada beneficio debe mostrar cómo mejora mi operación.
- Diseño ordenado y limpio.


### 4. Historia de Usuario – Sección Tecnología
Como visitante técnico,
quiero ver qué tecnología usa el sistema,
para saber si cumple con estándares modernos.

**Criterios de aceptación:**
- Debe incluir una imagen representativa.
- Debe explicar de forma sencilla la tecnología usada.
- Debe ser una sección visual y moderna.


### 5. Historia de Usuario – Sección Contacto
Como posible cliente,
quiero un formulario rápido para enviar mis datos,
para recibir más información o una demo.

**Criterios de aceptación:**
- Formulario con nombre, correo, empresa y mensaje.
- Botón para enviar.
- Diseño limpio y confiable.
